# Eye-contact

Exercise in using the Document Object Model and simple CSS to create code that reacts to user input.


Installation
------------------------------------------------
No installation needed

Usage
------------------------------------------------
Drop bigEyes.html into browser and thats it!


Roadmap
-------------------------------------------------

- update textures

Notes
----------------------------------------
